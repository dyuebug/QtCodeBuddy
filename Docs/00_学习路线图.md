# Qt6 学习路线图

## 一、项目简介

**QtCodeBuddy** 是一个 Qt6 学习项目，通过 12 个循序渐进的章节，帮助你从零开始掌握 Qt6 桌面应用开发。最终目标是完成一个功能完整的 **NotepadPro 个人笔记本应用**。

## 四、学习建议

### 4.1 学习顺序

**强烈建议按章节顺序学习**，因为：
- 每章都建立在前一章的基础上
- 知识点有明确的依赖关系
- 代码复杂度逐步递增

### 4.2 学习方法

1. **先读文档**：理解"是什么"和"为什么"
2. **再看代码**：理解"怎么用"
3. **动手实践**：修改代码，观察效果
4. **总结归纳**：整理自己的笔记

### 4.3 环境准备

| 工具 | 版本要求 |
|------|----------|
| Qt | 6.x（推荐 6.5+） |
| 编译器 | MSVC 2019+ / GCC 11+ / Clang 14+ |
| IDE | Qt Creator / VS Code / CLion |
| CMake | 3.16+ |

## 七、快速开始

### 7.1 编译运行第一个示例

```bash
# 进入第1章目录
cd Chapter01_Basic

# 创建构建目录
mkdir build && cd build

# 配置项目
cmake ..

# 编译
cmake --build .

# 运行
./Chapter01_Basic
```

### 7.2 遇到问题？

1. 确保 Qt6 已正确安装并配置环境变量
2. 检查 CMakeLists.txt 中的 Qt 路径
3. 查看各章节的 README 文件
